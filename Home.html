Project ‘Netra’

Our team has designed an AI based Surveillance-System which captures the movements and posture of individuals and deploys machine learning algorithms on them. With the help of this dataset the system maps out a certain pattern and then compares it with predefined patterns of human activity like sitting, standing, running etc. Moreover, it analyzes the activity in process and alerts the concerned authorities if some abnormal activity is found.
